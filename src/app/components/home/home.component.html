This application is used for practicing Angular code.

It will enable you to connect to a SQL Server database via a web Api, so you will be able to read and
list database information and also write data to a database using forms via the web Api.

The WebApi the application connects to is written in C# and uses Entity Framework as well, it enables CRUD operations
to be executed within the application and these will be tested.

The list of items to view are:

  • Sugar Readings
  • Addresses
  • HouseHold Bills
  • Pensions

Each Section will allow the input of new data using Angular forms

This web application will also incorporate and use Angular Materials where possible eventually.
